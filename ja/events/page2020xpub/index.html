〈ウェブ出版と日本語組版の未来〉について

〈ウェブ出版〉とは何でしょう？

  • ウェブ出版 vs 印刷出版
    • 紙とウェブの違い
      • 「紙はWebよりも難解な内容を、より深く長く論じることに向いたメディア」
      • 紙とウェブの組版スタイルの違い
        • ページめくり / スクロール
        • 固定ページサイズ＋文字数・行数 / 可変ページサイズ
        • 行末を揃える / 行末を揃えない
        • 段落区切りは段落先頭の1字下げ / 段落間のアキ
        • 縦書き / 横書き
      • 紙とウェブでそれぞれに向く出版コンテンツが違うのは、UIや組版スタイルの影響もあり？
        • 現状のウェブの組版スタイルは、現状のブラウザで使えるウェブ標準技術（とくに CSS）の制約も関係
          • ウェブ標準が進化することで変わってくる可能性も
  • EPUB・電子書籍
    • EPUB = ウェブ標準技術（HTML や CSS）を使った電子書籍フォーマット
      • EPUB 3.0 (2011年) で、縦書きなど日本語組版に必要な機能が標準に
        • 世界標準フォーマットでの日本語電子書籍が普及
        • W3C で関連する CSS 標準化（縦書き＝CSS Writing Modes など）が進んだことで実現
        • 「日本語組版処理の要件(JLREQ)」が役に立った！
    • EPUB は〈ウェブ出版〉といえるか？
      • ウェブの技術が使われているが、現状は、いろいろ違う

ウェブ出版と日本語組版の未来

  • ウェブ出版物 (Web publications) 関連の W3C 標準化動向
    • ウェブ上で本のようにも読める出版物の標準を作る動き
      • ウェブ出版物仕様（未完成でWGノートに） Web Publications (W3C WGノート)
        • 将来の EPUB 4.0 は、このウェブ出版物のパッケージ化形式になる予定だったが、先行きは不透明
      • 出版物の内容とメタ情報を JSON-LD 形式で記述 Publication Manifest (W3C 勧告候補)
    • 出版コンテンツの要素をセマンティックに Digital Publishing WAI-ARIA (W3C 勧告)
    • 出版物のテキストレイアウトとページレイアウトの要求からの CSS 標準化
      • 「日本語組版処理の要件(JLREQ)」をはじめ世界中の組版処理の要件
      • CSS でのページネーション
        • CSS Paged Media 関連はドラフト仕様はいろいろあるがブラウザへの実装は進んでいない
    • 〈ウェブ出版物〉が進化するとウェブ出版と EPUB 電子出版と印刷出版がシームレスに？
      • ウェブで公開してブラウザで読める
      • パッケージ化して EPUB 電子書籍になる
      • CSS 組版によって、印刷出版にもなる
  • 未来は
    • ウェブ出版が進化することで、その領域が広がって、印刷出版の領域とも重なっていくだろう
      • CSS 標準化が進むことで、ウェブ出版における日本語組版が進化
    • しかし、標準化が進むにはとても時間がかかる

CSS 組版と Vivliostyle

CSS 組版とは

  • ウェブと同様に HTML + CSS を使って、印刷出版物にもなる組版を行う技術
  • CSS 組版のメリット
    • HTML + CSS が使える人には簡単
      • InDesign や TeX など専用の組版ソフトウェアを習得する必要ない
    • 自動化しやすい
      • 例: 簡易マークアップ（マークダウン）から HTML への自動変換 → HTML + CSS からの PDF への自動 CSS 組版
    • ワンソース・マルチユースを実現できる
      • ウェブ出版、電子出版、印刷出版をすべて HTML + CSS で
        • それぞれのメディア用に調整すればよい
    • 出版物のアクセシビリティーの向上につながる
      • 出版物のソースデータが HTML などテキストデータ
      • 弱視者向けの大活字本を作るのもスタイルシートの調整で
  • ページ組版関連の CSS ドラフト仕様
    • CSS Paged Media
    • CSS Fragmentation
    • CSS Generated Content for Paged Media
    • CSS Generated Content
    • CSS Page Floats
